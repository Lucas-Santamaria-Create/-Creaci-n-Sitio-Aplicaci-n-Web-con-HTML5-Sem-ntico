Introducción a SQL

SQL (Structured Query Language) es un lenguaje de programación utilizado principalmente para gestionar bases de datos relacionales. Surgió en la década de 1970 como una solución para simplificar el acceso y manipulación de grandes cantidades de datos. SQL es crucial en sectores como:

  • Analítica de Datos: Usado para analizar grandes volúmenes de datos de manera eficiente.
  • Desarrollo Web: Permite que aplicaciones web almacenen y recuperen datos de usuarios.
  • Aplicaciones Empresariales: Facilita la gestión de información de clientes, productos y ventas.

Elementos Fundamentales

En SQL, trabajamos principalmente con los siguientes elementos:

  • Tablas: Estructuras que organizan datos en filas y columnas. Cada tabla en una base de datos representa una entidad específica, como Clientes o Productos.
  • Columnas: Representan los atributos de la entidad. En una tabla de Clientes, podrías tener columnas como Nombre, Edad, y Email.
  • Filas: Son los registros individuales de una tabla. Cada fila contiene datos específicos para un cliente o producto.
  • Tipos de Datos: Define qué tipo de información se puede almacenar en una columna. Algunos tipos comunes son:
    • INT: Números enteros.
    • VARCHAR: Cadenas de texto de longitud variable.
    • DATE: Fechas en formato AAAA-MM-DD.

Consultas Básicas

Las consultas en SQL nos permiten recuperar y organizar datos de acuerdo a criterios específicos. Estas son algunas de las instrucciones fundamentales:

  • SELECT: Selecciona columnas específicas de una tabla para mostrar los datos.
  • FROM: Indica de qué tabla se extraerán los datos.
  • WHERE: Filtra los resultados según una condición. Solo se muestran las filas que cumplan con el criterio.
  • ORDER BY: Organiza los resultados en un orden específico, ya sea ascendente (ASC) o descendente (DESC).

Ejemplo de una consulta básica:

SELECT nombre, edad FROM clientes WHERE edad > 18 ORDER BY nombre ASC;

Esta consulta selecciona los campos nombre y edad de la tabla clientes donde la edad es mayor a 18, y los ordena alfabéticamente por nombre.

Operaciones CRUD Básicas

CRUD es un acrónimo de Crear, Leer, Actualizar y Eliminar, las cuatro operaciones fundamentales para manipular datos en SQL:

  • INSERT: Agrega nuevos datos en una tabla. Ejemplo:
    • INSERT INTO clientes (nombre, edad) VALUES ('Juan', 25);

    Este comando añade un nuevo cliente llamado "Juan" de 25 años a la tabla clientes.

  • SELECT: Ya lo vimos antes, permite leer datos de una tabla.
  • UPDATE: Modifica datos existentes en la tabla. Ejemplo:
    • UPDATE clientes SET edad = 26 WHERE nombre = 'Juan';

    Este comando actualiza la edad de Juan a 26 en la tabla clientes.

  • DELETE: Elimina datos de una tabla. Ejemplo:
    • DELETE FROM clientes WHERE nombre = 'Juan';

    Este comando borra a Juan de la tabla clientes.

Script SQL para Crear la Tabla

Para crear la tabla clientes y añadir algunos registros iniciales, puedes usar el siguiente script SQL:

                -- Crear la tabla de clientes
CREATE TABLE clientes (
    id INT PRIMARY KEY AUTO_INCREMENT,
    nombre VARCHAR(50) NOT NULL,
    edad INT,
    email VARCHAR(100)
);

-- Insertar algunos registros iniciales en la tabla clientes
INSERT INTO clientes (nombre, edad, email) VALUES 
    ('Juan', 25, 'juan@example.com'),
    ('Maria', 30, 'maria@example.com'),
    ('Carlos', 22, 'carlos@example.com'),
    ('Ana', 28, 'ana@example.com');

Utiliza la tabla anterior para realizar los siguientes ejercicios:

  1. Selecciona todos los registros de la tabla clientes. 
    SELECT * FROM clientes;
  2. Inserta un nuevo cliente llamado "Luis" con 35 años de edad. 
    INSERT INTO clientes (nombre, edad, email) VALUES ('Luis', 35, 'luis@example.com');
  3. Actualiza la edad del cliente llamado "Juan" a 28 años. 
    UPDATE clientes SET edad = 28 WHERE nombre = 'Juan';
  4. Elimina el registro del cliente llamado "Maria". 
    DELETE FROM clientes WHERE nombre = 'Maria';
